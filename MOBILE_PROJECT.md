# 移动端项目标识

## 项目信息
- **项目类型**: 移动端官网（Mobile Web App）
- **目标设备**: 智能手机（iOS/Android）
- **视口宽度**: 390px (iPhone 12/13 参考尺寸)
- **设计稿来源**: Sketch 设计稿（已提取至 sketch_extract/）

## 关键特征
- ✅ 响应式设计（移动优先）
- ✅ 触控交互优化
- ✅ 底部 TabBar 导航
- ✅ 移动端原生体验（Lenis 平滑滚动）
- ✅ 本地存储（localStorage）用于状态管理

## 技术约束
- 不依赖桌面端布局
- 所有交互针对触摸屏设计
- 性能优化关注移动网络环境
- 字体大小、按钮尺寸适合手指操作

## 开发规范
1. 始终使用移动端视口测试（建议 390x844）
2. 检查横向滚动溢出
3. 确保触摸目标 ≥ 44×44px
4. 使用 system fonts 减少加载
